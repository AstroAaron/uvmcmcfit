Installation
************

#. Install Python. I recommend the `Anaconda python distribution <https://store.continuum.io/cshop/anaconda/>`_.  
#. Install `pyyaml <http://pyyaml.org/>`_ (pip install pyyaml)
#. Install `emcee <http://dan.iel.fm/emcee/current/>`_ (pip install emcee)
#. Install ``uvmcmcfit``

   * Download ``uvmcmcfit`` from github: https://github.com/sbussmann/uvmcmcfit
   * Add the directory containing ``uvmcmcfit`` to your PYTHONPATH
